# ContestHUB

A platform for creators and participants to manage, join, and compete in online contests.

**Live Site:** https://contest-hub-988e2.web.app

---

## Features

- **User Authentication:** Secure signup and login using Firebase.
- **Contest Creation:** Creators can submit contests for approval.
- **Contest Participation:** Users can register and pay to participate in contests via Stripe.
- **Submissions:** Participants can submit entries for contests directly on the platform.
- **Winner Management:** Creators can declare winners; users cannot register for closed contests.
- **Responsive Design:** Fully responsive UI for desktop, tablet, and mobile devices.
- **Interactive Charts:** Visualize contest statistics with Recharts.
- **Animated UI:** Smooth animations using GSAP for better user experience.
- **Notifications:** Instant feedback on registration, submission, and payment using React Toastify.
- **Search & Filter:** Easily search and filter contests by type and status.
- **Countdown Timer:** Real-time countdown for contest deadlines.
- **Admin Panel:** Admins can approve contests and manage users and creators.

---

## Technologies & Libraries Used

- **Frontend:** React, Tailwind CSS, DaisyUI
- **Routing:** React Router
- **Forms & Validation:** React Hook Form
- **State Management & Data Fetching:** React Query, Axios
- **Animations:** GSAP, Typewriter Effect
- **Notifications:** React Toastify, SweetAlert2
- **Charts & Analytics:** Recharts
- **Carousel & Marquee:** React Slick, React Fast Marquee, Slick Carousel
- **Icons:** React Icons, Lucide React
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** Firebase
- **Payments:** Stripe API
- **Date Handling:** date-fns
- **Build Tool:** Vite
